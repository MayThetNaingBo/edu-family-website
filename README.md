## Client Project

This website was developed for **EDU Family Private High School**, an operational private high school in Myanmar.

The objective was to provide the school with a modern, bilingual, and responsive website that effectively communicates information to prospective students, parents, and the wider community.

> This repository is shared for portfolio purposes with the client's permission. Sensitive information and credentials are not included.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
